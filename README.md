# AppWeatherMovil

1. Nombre del proyecto: AppMovilClima

2. Integrantes: Martin Carrasco y Victor Araujo

3. La app permite buscar una ciudad y al buscarla esta te otorga informacion sobre el clima y las temperaturas actuales de esta misma ademas que las ciudades buscadas se guardan en un historial al cual se puede acceder mediante un boton y ahi se puede observar las ciudades buscadas mostrando la hora a la que fue buscada junto con los datos del clima de ese momento

4. API utilizada: OpenWeatherMap para obtener los datos en tiempo real de las ciudades buscadas

5. para ejecutar el proyecto es recomendable instalar el plugin "JSON to Kotlin class" y al estar instalado y el proyecto haya finalizado su compilacion solo con darle play al app funcionar√°
